Elasticsearch driver [Dobo](https://github.com/ardhi/dobo)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install dobo-elasticsearch
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```dobo-elasticsearch``` in it
